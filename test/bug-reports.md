# 🐛 BUG-001: Отсутствие урона от отравляющего озера

| Параметр | Значение |
| :--- | :--- |
| **Серьёзность** | Major |
| **Приоритет** | High |
| **Статус** | Открыт |
| **Предусловие** | Игрок находится в зоне отравляющего озера |
| **Шаги воспроизведения** | 1. Подойти к отравляющему озеру<br>2. Войти в воду<br>3. Оставаться в зоне несколько секунд |
| **Ожидаемый результат** | Игрок получает урон от отравления |
| **Фактический результат** | Урон не наносится |

# 🐛 BUG-002: Босса можно убить до начала боя заклинанием “Сосулька”

| Параметр | Значение |
| :--- | :--- |
| **Серьёзность** | Critical |
| **Приоритет** | High |
| **Статус** | Открыт |
| **Предусловие** | Игрок находится перед входом в локацию босса |
| **Шаги воспроизведения** | 1. Подойти к входу в арену босса<br>2. Использовать заклинание “Сосулька”<br>3. Попасть по боссу до активации боя<br>4. Убить босса |
| **Ожидаемый результат** | Босс активируется только внутри арены и не может быть убит до начала боя |
| **Фактический результат** | Босса можно убить заранее и получить награду без сражения |

📎 **Доказательства:**
- 📽️ Смотреть видео: - https://youtu.be/AFxvbjJsAqc

# 🐛 BUG-003: Заклинание “Сосулька” мгновенно убивает всех врагов

| Параметр | Значение |
| :--- | :--- |
| **Серьёзность** | Critical |
| **Приоритет** | High |
| **Статус** | Открыт |
| **Предусловие** | Начало боя с несколькими врагами |
| **Шаги воспроизведения** | 1. Вступить в бой с врагами<br>2. Использовать заклинание “Сосулька”<br>3. Наблюдать результат |
| **Ожидаемый результат** | Заклинание наносит сбалансированный урон согласно механике игры |
| **Фактический результат** | Все враги погибают мгновенно независимо от здоровья |

# 🐛 BUG-004: При использовании способности «Пинок» можно видеть сквозь стены

| Параметр | Значение |
| :--- | :--- |
| **Серьёзность** | Medium |
| **Приоритет** | Medium |
| **Статус** | Открыт |
| **Предусловие** | Игрок находится рядом со стеной или интерактивным объектом возле стены |
| **Шаги воспроизведения** | 1. Подойти вплотную к стене.<br>2. Зажать клавишу **E** для использования способности «Пинок».<br>3. Во время анимации пинка продолжать прижиматься к объекту.<br>4. Повернуть камеру в сторону стены. |
| **Ожидаемый результат** | Камера не проходит сквозь геометрию уровня, обзор ограничивается стеной. |
| **Фактический результат** | Камера проходит сквозь стену, что позволяет видеть помещения и объекты за пределами локации. |

📎 **Доказательства:**
- 📽️ Смотреть видео: https://youtu.be/kqYyCLWrb9s

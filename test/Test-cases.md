# 🧪 Тест-кейсы — Whatever It Takes (Unreal Engine 5)

## 🎮 Описание
Тест-кейсы составлены для проверки игровых механик в проекте **Whatever It Takes** от студии **Next Genium**.  
Основной фокус тестирования: взаимодействие с объектами, использование заклинаний и боевая система.

---

## TC-001: Активация рычага

| Параметр | Значение |
| :--- | :--- |
| **Приоритет** | Высокий |
| **Статус** | Не выполнен |
| **Предусловие** | Игрок находится рядом с рычагом |
| **Шаги выполнения** | 1. Подойти к рычагу<br>2. Навести камеру на рычаг<br>3. Нажать клавишу взаимодействия |
| **Ожидаемый результат** | Рычаг активируется, запускается связанное событие: открытие двери |

## Примечание

Данные тест-кейсы покрывают основные игровые механики:
- взаимодействие с объектами;
- использование рычагов;
- применение магии;
- проверку боевой системы;
- убийство монстров;
- работу эффектов и способностей.
